# Nazar_Balloon landing page

Готовий статичний сайт для GitHub Pages.

## Структура

- `index.html` — головна сторінка
- `style.css` — стилі
- `script.js` — мобільне меню
- `images/` — папка для фотографій

## Які фото додати

Збережіть свої фото з Instagram у папку `images` з такими назвами:

- `hero.jpg`
- `category01.jpg` ... `category06.jpg`
- `product01.jpg` ... `product04.jpg`
- `gallery01.jpg` ... `gallery04.jpg`

## Що замінити

У `index.html` замініть:

- `+38 000 000 00 00` на реальний телефон
- посилання `href="#"` біля Telegram на реальне посилання, наприклад `https://t.me/username`
- ціни та описи товарів за потреби

## GitHub Pages

1. Створіть репозиторій.
2. Завантажте всі файли в корінь репозиторію.
3. Settings → Pages → Deploy from branch → main → root.
4. Через кілька хвилин сайт буде доступний за адресою GitHub Pages.
